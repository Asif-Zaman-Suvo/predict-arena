# Tasks: FIFA World Cup 2026 Predictor — Frontend

**Scope:** Layout · Landing · Groups · Bracket · Leaderboard · Community · Profile
**Spec:** `docs/SPEC.md` · **Plan:** `docs/PLAN.md`
**Date:** 2026-05-30

---

## Scope Notes

Two pages in this breakdown are **new additions** not in the original spec:

- `/community` — Community predictions feed: mock activity, community pick percentages per match, top predictors.
- `/profile` — User profile: display name, avatar, prediction stats, edit name.

The **bracket page** in this scope is read-only. It is pre-seeded with `data/sample-predictions.json` so it renders a full interesting bracket without requiring the group prediction entry flow. The interactive `/predict/knockout` page is out of scope here.

The **groups page** derives standings from `sample-predictions.json` by default, so standings always show meaningful data on first visit.
